# Code of Conduct

## Our pledge

This project is committed to providing a welcoming, harassment-free experience
for everyone who participates, regardless of background or identity. We expect
all contributors and maintainers to interact respectfully and in good faith.

## Standard

terrastrata adopts the [Contributor Covenant][homepage], version 2.1, as its code
of conduct. Please read the full text here:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be considerate and constructive, assume good intent, accept feedback
gracefully, and focus on what is best for the community. Behavior that the
Contributor Covenant identifies as unacceptable will not be tolerated.

## Scope

This Code of Conduct applies within all project spaces — issues, pull requests,
discussions, and code review — and when an individual is representing the project
in public spaces.

## Enforcement

Reports of unacceptable behavior can be made privately to the maintainer at
**pascal.toepke@gmail.com**. All reports will be reviewed and investigated
promptly and fairly, and the maintainer will respect the privacy and security of
the reporter. Maintainers who do not follow or enforce this Code of Conduct in
good faith may face repercussions as determined by the project's leadership.

For the enforcement guidelines and the full set of community-impact remedies, see
the Contributor Covenant text linked above.

[homepage]: https://www.contributor-covenant.org
